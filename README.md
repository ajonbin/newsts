# Prompt #

请获取2024巴黎奥运会的五条最新动态，按照新闻热度排序，并形成摘要，每条新闻的摘要要求不少于500字。

请将上述摘要中的阿拉伯数字用中文字表示

# Usage #

~~~
$ python newsts.py --help
usage: newsts.py [-h] [-f NEWS_FILE]

News To Speech

options:
  -h, --help            show this help message and exit
  -f NEWS_FILE, --news-file NEWS_FILE
                        Text file include news lines
